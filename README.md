# Election Assistant

An interactive, SPA (Single Page Application) built with vanilla HTML/CSS/JS and Node.js. It features a Voting Booth secured by Face-API biometric authentication, an Admin Portal to handle candidates, and a dedicated AI Assistant powered by the Gemini API.

## Features
- **Admin Portal**: Password-protected area to add election candidates dynamically.
- **Biometric Voting Booth**: Uses client-side `face-api.js` to ensure unique identity verification before casting a vote.
- **AI Chat Assistant**: Integrated with Gemini 2.5 Flash API to answer civic and democratic process queries neutrally.
- **Election Information**: Timelines, processes, and FAQs right in the sidebar.
